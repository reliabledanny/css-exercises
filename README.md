# CSS Exercises

This project is part of The Odin Project and focuses on learning the practice of adding CSS to an HTML file.

The goal of this project is to apply CSS styles using all three methods: external CSS, internal CSS, and inline CSS.

By completing this project, I will practice utilizing CSS in web pages, making the web pages more appealing.

## Skills demonstrated
By completing this project, I will have demonstrated the ability to:

- Apply CSS to HTML file
- Use the appropriate selectors
- Design webpage that is visually appealing
